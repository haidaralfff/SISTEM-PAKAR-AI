# Prompt UI Dashboard - Ruang Pulih (Mental Health SaaS)

## Objective

Design a modern SaaS dashboard for a mental health platform called **Ruang Pulih**. The interface should feel calm, trustworthy, clean, and emotionally supportive. Use a premium SaaS aesthetic inspired by Linear, Stripe Dashboard, Notion, and Calm.

---

## Design Style

- Modern SaaS Dashboard
- Minimalist
- Soft rounded corners (16–20px)
- Spacious layout
- Clean typography
- Soft shadows
- Large white cards
- Neutral background (#F8FAFC)
- Primary color: Sage Green (#6FAF8F)
- Secondary: Soft Cream (#FAF8F3)
- Accent: Emerald (#4E7A65)
- Border: #E5E7EB

---

## Overall Feel

The dashboard should feel like:

- Calm
- Premium
- Professional
- Friendly
- Safe
- Modern
- Comfortable
- Easy to read

Avoid looking like a banking dashboard.

Instead, create a wellness-focused SaaS dashboard that encourages users to continue their mental health journey.
