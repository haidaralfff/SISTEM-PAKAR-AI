# Dashboard UI Prompt

## Objective
Design a modern, clean, and minimal analytics dashboard for a web application. The interface should have a professional SaaS appearance with a white background, subtle shadows, rounded corners, and a green accent color.

---

## Layout

The dashboard consists of three sections:

1. Collapsible icon sidebar (left)
2. Navigation sidebar
3. Main dashboard content

Use a desktop-first responsive layout with a width of approximately 1440px.

---

## Design Style

- Modern SaaS Dashboard
- Minimalist
- Soft shadows
- Rounded corners (12–16px)
- White cards
- Light gray background (#F8F9FB)
- Primary color: Emerald Green (#16A34A)
- Secondary colors:
  - Blue (#3B82F6)
  - Orange (#F59E0B)
  - Gray (#6B7280)

Typography:
- Font: Inter
- Clean spacing (24px)
- Consistent padding
